# DAY 4

This day is about the basics of package management. I chose to play around with the "ncdu" package. \
I didn't have it installed so I decided to inspect it and explore more on storage. 
https://imgur.com/a/wEbyIew - by searchning for the package naming we can choose the specific one we want. \
After installing it is a matter of getting used with it (if not already) I browsed throgh the man pages of the package. \
For example "ncdu -x / " using it like this would start browsing from our root directory and from there we can analyze whatever we feel like. Not the most optimal path, but we also have to dive into standard directories and also review configuration files. - https://imgur.com/a/gnnOOcH \
Getting into my /var/log dir is super easy this way. It was a matter of 2 enters and a few arrow key scrolls. I heard about this tool recently and am liking how minimalistic yet fast it feels to browse. \
Also reviewed "mc", I've used it a few times before. I like to use it to view files directly, you can do that by pressing F3 on a selected file. Then F10 closes \
In my case the populated sources list is under /etc/apt/sources.list directly - https://imgur.com/vebGqPw \
As far as I am aware sources under "sources.list.d" are third party(docker I believe is stored here) and all standard ones are still within the usual sources.list file.
