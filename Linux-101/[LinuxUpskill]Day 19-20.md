# DAY 19

This day is about how files actually work underneath in Linux. We look at inodes first, since every file is tied to one. then we create hard links and symlinks to see the difference in how they behave.
After that we check how these links show up with ls and stat, and also look at real examples in the system.
At the end we create a simple alias(I will modify a one I use), just as a quick way to make commands easier to use.
